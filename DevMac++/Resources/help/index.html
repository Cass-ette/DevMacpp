DevMac++ 使用指南

文件操作

新建文件：点击工具栏 doc 图标或按 Cmd+N，在弹出的模板选择器中选择模板。

保存模板：在模板选择器底部点击"将当前代码保存为模板..."，可将当前代码保存为自定义模板；内置模板不可删除，自定义模板右侧有 × 删除按钮。

多窗口：每个窗口独立管理一个文件，编译器、调试器互不干扰。

编译与运行

编译：按 Cmd+B 或工具栏锤子按钮，编译日志显示在底部面板。

运行：按 Cmd+F9 或工具栏橙色播放按钮，自动编译并运行。

粘贴测试输入：程序运行后，将样例输入（含换行）粘贴进底部"运行"面板的多行输入框，点击"发送"即可一次性喂给程序，无需手动逐行输入。程序未到达读取语句时发送也有效，数据会缓存在管道里等待读取。

查找与替换

按 Cmd+F 在编辑器内弹出查找框，支持大小写匹配。Enter / Shift+Enter 切换上下匹配项，Escape 关闭。

点击查找框左侧箭头展开替换输入框，支持替换当前和全部替换。

调试器

设置断点

在编辑器行号左侧单击即可切换断点（红色圆点）。可以在程序运行时设置断点，调试器会在程序下次暂停时批量应用。

调试流程

1. 在需要的行设好断点，按 Cmd+F8 开始调试

2. 程序运行到断点时自动暂停，编辑器高亮当前行

3. 左侧侧边栏显示：监视变量（可添加任意表达式）、局部变量（当前作用域所有变量）、调用栈

4. 使用单步快捷键逐行执行，按 Cmd+F3 继续运行到下一断点

监视变量

在左侧"监视"面板点击 + 按钮，输入任意 C++ 表达式（变量名、a[i]、a+b 等）。每次暂停时自动求值更新，支持 vector、数组、结构体等复合类型。

调试期间输入

程序需要 stdin 输入时，在底部"调试"面板的输入框中填写后按回车发送，再按"继续"让程序继续执行。

调试器基于 LLDB。调试编译会自动加入 -g -O0 -fno-inline 选项，确保断点能正确定位到 struct/class 内联方法。

容器 (Containers)

序列容器

vector<T> - 动态数组，支持随机访问

deque<T> - 双端队列

list<T> - 双向链表

array<T, N> - 固定大小数组

关联容器

set<T> - 有序集合（红黑树）

map<K, V> - 有序键值对

unordered_set<T> - 哈希集合

unordered_map<K, V> - 哈希映射

multiset<T> / multimap<K,V> - 可重复版本

常用操作

v.push_back(x) - 末尾添加

v.size() - 获取大小

v.empty() - 判断是否为空

v.clear() - 清空

v.begin() / v.end() - 迭代器

v.erase(it) - 删除

算法 (Algorithms)

排序与搜索

sort(v.begin(), v.end()) - 排序（平均 O(n log n)）

stable_sort() - 稳定排序

reverse(v.begin(), v.end()) - 反转

unique(v.begin(), v.end()) - 去重（需配合 erase）

binary_search(v.begin(), v.end(), x) - 二分查找

lower_bound(v.begin(), v.end(), x) - 第一个 >= x 的位置

upper_bound(v.begin(), v.end(), x) - 第一个 > x 的位置

数值运算

accumulate(v.begin(), v.end(), init) - 求和

max_element(v.begin(), v.end()) - 最大值迭代器

min_element(v.begin(), v.end()) - 最小值迭代器

查找与计数

find(v.begin(), v.end(), x) - 查找，返回迭代器

count(v.begin(), v.end(), x) - 计数

any_of() / all_of() / none_of() - 断言检查

字符串 (Strings)

string 常用方法

s.length() / s.size() - 长度

s.substr(pos, len) - 子串

s.find(str) - 查找子串（返回位置或 npos）

s.replace(pos, len, str) - 替换

s.erase(pos, len) - 删除

s.insert(pos, str) - 插入

to_string(x) - 任意类型转字符串

stoi(s) / stoll(s) / stod(s) - 字符串转数值

stringstream ss(s); ss >> x; - 字符串流解析

字符判断

isdigit(c) / isalpha(c) / isalnum(c)

isspace(c) / isupper(c) / islower(c)

输入输出 (I/O)

C++ 流

cin >> x - 读取（跳过空格、换行）

cout << x - 输出

getline(cin, s) - 读取整行（含空格）

cin.get() - 读取单个字符

cin.ignore(n, delim) - 忽略字符

C 风格（竞赛常用，快于 cin/cout）

scanf("%d", &x) - 格式化输入

printf("%d\n", x) - 格式化输出

gets(s) - 读取一行（已废弃）

加速 IO（必须加！）

ios::sync_with_stdio(false); - 关闭同步

cin.tie(nullptr); - 解除 cin 和 cout 绑定

竞赛提示：在 main 函数开头加这两行可以大幅加速 IO

工具类 (Utilities)

pair 和 tuple

pair<T1, T2> - 二元组

make_pair(a, b) - 创建 pair

p.first / p.second - 访问元素

tuple<T1, T2, T3> - 多元组（C++11）

get<i>(t) - 访问第 i 个元素

Lambda 表达式（C++11）

[](int x) { return x * 2; } - 基本形式

[&](int x) { return x + y; } - 按引用捕获

[=](int x) { return x + y; } - 按值捕获

[x, &y](int z) { } - 混合捕获

[=]() mutable { x++; } - mutable（修改捕获值）

STL 容器初始化（C++11）

vector<int> v = {1, 2, 3};

string s = "hello";

for (auto& x : v) { } - 范围 for 循环

编译选项
